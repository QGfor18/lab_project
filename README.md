# FPGA矩阵运算系统

## 项目概述
基于FPGA的矩阵运算系统，支持矩阵转置、加法、标量乘法、矩阵乘法以及卷积运算。

## 项目结构

```
lab_project/
├── src/                    # 源代码目录
│   ├── matrix_core/        # 矩阵运算核心模块
│   ├── input_interface/    # 输入接口模块
│   ├── output_interface/   # 输出接口模块
│   ├── storage/           # 存储管理模块
│   ├── control_fsm/       # 控制状态机
│   └── top/               # 顶层模块
├── sim/                   # 仿真目录
│   ├── test_vectors/      # 测试向量
│   └── scripts/           # 仿真脚本
├── constraints/           # 约束文件目录
├── doc/                   # 文档目录
└── tools/                 # 工具目录
    ├── python/            # Python串口工具
    └── java/             # Java串口工具
```

## 功能特性

### 核心功能
- 矩阵输入/生成/存储/展示
- 矩阵转置运算
- 矩阵加法运算
- 标量乘法运算
- 矩阵乘法运算

### 附加功能
- 卷积运算（Bonus）
- 动态参数配置
- 串口可视化工具

### 硬件接口
- UART串口通信
- 按键输入
- 拨码开关
- LED显示
- 数码管显示

## 技术规格
- 最大矩阵尺寸：5x5
- 元素范围：-128 ~ 127（8位有符号数）
- 支持矩阵存储数量：可配置（默认2个）
- 运算精度：整数运算

## 快速开始
1. 打开Vivado项目文件 `lab_project.xpr`
2. 添加源文件到相应目录
3. 配置约束文件
4. 运行仿真测试
5. 综合并生成比特流
